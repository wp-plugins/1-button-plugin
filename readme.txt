== Description ==



This plugin adds a google +1 button to your site. +1 buttons let people who love your content recommend it on Google search.


= Features =


adds a google +1 button to your site.



== Installation ==


1. Upload the whole plugin folder to the `/wp-content/plugins/` directory

2. Activate the plugin through the 'Plugins' menu in WordPress.

3. Use the 'Options' page to change your plugin options.
